<span underline="single">Se connecter à un poste de travail</span>

Une fois que vous êtes connecté à View Connection Server et que votre session est ouverte, vous pouvez vous connecter à un poste de travail virtuel. La fenêtre View Client affiche une liste des postes de travail que vous êtes autorisé à utiliser et l'état actuel de chaque poste de travail.

Pour vous connecter à un poste de travail :
1. Sélectionnez un poste de travail dans la liste dans la fenêtre View Client.
2. (Facultatif) Sélectionnez une option dans le menu déroulant <b>Display (Affichage)</b>. Plein écran permet d'afficher le poste de travail sur l'intégralité de l'écran. Si vous sélectionnez le mode Plein écran et que vous avez plusieurs écrans, faites glisser la fenêtre de sélection des postes de travail vers l'écran dans lequel vous souhaitez afficher le poste de travail.
3. (Facultatif) Selon la façon dont votre administrateur View a configuré le système, vous pouvez sélectionner un protocole d'affichage. Cliquez sur la flèche vers le bas, sélectionnez <b>Protocols (Protocoles)</b> et sélectionnez Microsoft RDP ou PCoIP. Pour des instructions spécifiques, contactez votre administrateur View.
4. Double-cliquez sur le poste de travail dans la liste, cliquez sur <b>Connect (Se connecter)</b> ou cliquez sur la flèche vers le bas et sélectionnez <b>Connect (Se connecter)</b>.

Vos sélections sont enregistrées comme préférences pour votre prochaine connexion.

<span underline="single">Restaurer un poste de travail</span>

Vous pouvez restaurer un poste de travail virtuel désactivé sur un autre hôte. La restauration d’un poste de travail désactivé ignore les modifications faites sur le poste de travail, et toutes les données non enregistrées sur le serveur sont perdues.

Pour restaurer un poste de travail :
1. Cliquez sur la flèche vers le bas à côté du poste de travail désactivé que vous voulez restaurer.
2. Sélectionnez <b>Rollback (Restaurer)</b>.

La désactivation sur l'hôte distant est annulée et vous pouvez maintenant ouvrir une session sur le poste de travail sur l’hôte local.

<span underline="single">Se déconnecter d'un poste de travail</span>

Lorsque vous vous déconnectez d’un poste de travail virtuel, votre session reste ouverte. Tous les programmes s’exécutant sur le poste de travail restent ouverts. Vous pouvez vous reconnecter au poste de travail à tout moment et reprendre votre session.

Pour vous déconnecter d’un poste de travail :
1. Enregistrez tous les fichiers ouverts sur le poste de travail.
2. Dans le menu <b>Start (Démarrer)</b> du poste de travail, sélectionnez <b>Disconnect (Se déconnecter)</b> ou appuyez sur <b>Ctrl+Alt+Suppr</b> et cliquez sur <b>Disconnect (Se déconnecter)</b>.

<span underline="single">Fermer une session sur un poste de travail</span>

Lorsque vous fermez votre session sur un poste de travail virtuel, la session actuelle se termine, tous les programmes sont fermés et toutes les données non enregistrées sont perdues. Vous pouvez fermer votre session sur le poste de travail ou depuis la fenêtre View Client.

Pour fermer votre session sur un poste de travail :
1. Enregistrez tous les fichiers ouverts sur le poste de travail.
2. Dans le menu <b>Start (Démarrer)</b>, sélectionnez <b>Log Off (Fermer la session)</b>.
3. Dans la boîte de dialogue de confirmation, cliquez sur <b>Log Off (Fermer la session)</b>.
Vous êtes invité à enregistrer tous les fichiers ouverts si vous ne l’avez pas déjà fait.

Pour fermer votre session depuis la fenêtre View Client :
1. Cliquez sur la flèche vers le bas à côté du poste de travail désactivé pour fermer la session.
2. Sélectionnez <b>Log Off (Fermer la session)</b>.

<span underline="single">Réinitialiser un poste de travail</span>

Lorsque vous réinitialisez un poste de travail, il est fermé puis redémarré, et toutes les données non enregistrées sont perdues. Votre session doit être ouverte sur un poste de travail pour le réinitialiser.

Vous pouvez réinitialiser un poste de travail depuis la fenêtre View Client ou sur le poste de travail lui-même.

Pour réinitialiser un poste de travail depuis la fenêtre View Client :
1. Dans la fenêtre View Client, cliquez sur la flèche vers le bas à côté du poste de travail à réinitialiser.
2. Sélectionnez <b>Reset (Réinitialiser)</b>.
3. Cliquez sur <b>Reset (Réinitialiser)</b> pour confirmer.

Pour réinitialiser un poste de travail sur le poste de travail :
1. Appuyez sur <b>Ctrl+Alt+Suppr</b>.
2. Cliquez sur <b>Reset (Réinitialiser)</b>.

<span underline="single">Actualiser la liste des postes de travail</span>

Vous pouvez actualiser la liste des postes de travail pour afficher les modifications faites depuis la dernière extraction de la liste de View Connection Server. Pour cela, appuyez sur la touche <b>F5</b>.

<span underline="single">View Connection Server への接続</span>

VMware View Client を使用して、使用を許可されている仮想デスクトップに接続できます。仮想デスクトップおよび仮想デスクトップにインストールされたソフトウェアについて詳しくは、View 管理者へお問い合わせください。

View Client は、View Connection Server に接続するネイティブ Linux アプリケーションとして動作します。ログインして仮想デスクトップに接続するには、View Connection Server への接続が必要です。

注意：View Client から接続できる View Connection Server のインスタンスは、一度に 1 つだけです。接続するには、サーバのホスト名または IP アドレスを指定できる必要があります。

View Connection Server に接続するには
1. View 管理者の指示に従って View Client を起動します。
2. [<b>アドレス</b>] ドロップダウン メニューから View Connection Server を選択するか、サーバのホスト名または IP アドレスを入力します。
3. （オプション）選択したサーバに接続するためのオプションを表示または非表示にするときは、[<b>オプション</b>] をクリックします。これらのオプションの構成は、View 管理者から指示された場合にのみ行ってください。オプションには以下のものがあります。

[<b>ポート</b>]　ポート エントリーは View Connection Server のデフォルト ポートに設定されています。
[<b>安全な接続（SSL）を使用</b>]　企業の機密情報を保護してすべての接続を完全に暗号化するために安全な（SSL）接続を使用するときに選択します。安全でない接続を設定する場合でも、常に安全な接続を使用するように View 管理者が View Connection Server を構成している場合があります。

注意：ポートおよび SSL オプションは、アドレス フィールドにそれらを直接入力することで指定することもできます。
非 SSL 接続の場合は、次を入力します：http://&lt;URL&gt;[:ポート番号]。SSL 接続の場合は、次を入力します：https://&lt;URL&gt;[:ポート番号]。

[<b>起動時に常にこのサーバに接続</b>]　View Client の起動時に View Connection Server の現在のインスタンスに直接接続するときに選択します。常に同じ View Connection Server に接続する場合は、このオプションを選択してください。選択しない場合は、View Client を起動するときに View Connection Server を選択するように求められます。

注意：接続に失敗した場合は、サーバ選択ダイアログ ボックスに戻ります。接続に成功した場合は、接続時に表示される最初のダイアログ ボックスで [<b>キャンセル</b>] をクリックすることで、サーバ選択ダイアログ ボックスに戻ることができます。

4. [<b>接続</b>] をクリックします。

<span underline="single">View Connection Server へのログイン</span>

View Connection Server に接続した後に、View 管理者から提供されたユーザー名とパスワードを使用してログインすることを求められます。システムの設定方法によっては、RSA SecurID、またはサポート対象のスマート カードを使用して、リモートでログインできる場合があります。これらのいずれかの方法でのログインに関する具体的な手順については、View 管理者へお問い合わせください。

ログインするには
1. View Connection Server に接続します。
2. ログイン ウィンドウで、ユーザー名とパスワードを入力します。
3. ドロップダウン メニューから適切なドメインを選択します。
4. [<b>ログイン</b>] をクリックします。
5. パスワードの変更を求められた場合は、古いパスワードを入力し、新しいパスワードを入力および確認入力し、[<b>変更</b>] をクリックすることにより、パスワードを変更します。

使用を許可されている仮想デスクトップのリストが表示されます。

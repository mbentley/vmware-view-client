<span underline="single">登录 View Connection Server</span>

当您连接到 View Connection Server 后，系统会提示您使用 View 管理员提供的用户名和密码进行登录。在一定的系统配置下，您还可以通过 RSA SecurID 或受支持的智能卡进行远程登录。有关如何使用上述方法进行登录的详细说明，请与您的 View 管理员联系。

要进行登录：
1. 连接到 View Connection Server。
2. 在登录窗口中输入您的用户名和密码。
3. 从下拉菜单中选择相应的域。
4. 单击<b>登录</b>。
5. 如果收到提示，请输入您的旧密码，然后输入并确认新密码，最后单击<b>更改</b>以更改密码。

屏幕上将显示您有权使用的虚拟桌面列表。

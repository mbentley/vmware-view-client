<span underline="single">デスクトップへの接続</span>

View 接続サーバに接続してログインした後に、仮想デスクトップに接続できます。View Client のウィンドウには、使用を許可されているデスクトップの一覧と、各デスクトップの現在の状態が表示されます。

デスクトップに接続するには
1. View Client ウィンドウ内のリストからデスクトップを選択します。
2. （オプション）[<b>ディスプレイ</b>] ドロップダウン メニューからオプションを選択します。[フル スクリーン] では、モニタ画面全体にデスクトップが表示されます。フル スクリーンを選択し、かつ複数のモニタがある場合、デスクトップを表示するモニタにデスクトップ選択ウィンドウをドラッグします。
3. （オプション）View 管理者によるシステムの設定方法により、ディスプレイ プロトコルを選択できます。下矢印をクリックして [<b>プロトコル</b>] を選択し、[Microsoft RDP] または [PCoIP] を選択します。具体的な手順については、View 管理者へお問い合わせください。
4. リスト内でデスクトップをダブルクリックし、[<b>接続</b>] をクリックするか、下矢印をクリックして [<b>接続</b>] を選択します。

選択した内容は、次に接続するときのために環境設定として保存されます。

<span underline="single">デスクトップのロールバック</span>

別のホストでチェックアウトした仮想デスクトップをロールバックできます。チェックアウト済みのデスクトップをロールバックすると、デスクトップに対して行われた変更はすべて破棄され、サーバにバックアップされていなかったデータはすべて失われます。

デスクトップをロールバックするには
1. ロールバックするチェックアウト済みデスクトップの横にある下矢印をクリックします。
2. [<b>ロールバック</b>] を選択します。

リモート ホストでのチェックアウトがキャンセルされ、ローカル ホストでデスクトップにログインできるようになります。

<span underline="single">デスクトップからの切断</span>

仮想デスクトップから切断しても、まだログインしたままです。デスクトップで実行されているプログラムがある場合は、それらも開いたままです。いつでもデスクトップに再接続してセッションをレジュームできます。

デスクトップ内から切断するには
1. デスクトップで開いているファイルがある場合は、それらを保存します。
2. デスクトップの [<b>スタート</b>] メニューから [<b>切断</b>] を選択するか、<b>&lt;Ctrl&gt; + &lt;Alt&gt; + &lt;Del&gt;</b> を押して [<b>切断</b>] をクリックします。

<span underline="single">デスクトップからのログオフ</span>

仮想デスクトップをログオフすると、現在のセッションは終了し、すべてのプログラムが閉じ、保存されていないデータはすべて失われます。ログオフは、デスクトップ内からまたは View Client ウィンドウから実行できます。

デスクトップ内からログオフするには
1. デスクトップで開いているファイルがある場合は、それらを保存します。
2. [<b>スタート</b>] メニューから [<b>ログオフ</b>] を選択します。
3. 確認ダイアログ ボックスで [<b>ログオフ</b>] をクリックします。
開いているファイルでまだ保存していないものがある場合は、保存することを求められます。

View Client ウィンドウ内からログオフするには
1. ログオフするチェックアウト済みデスクトップの横にある下矢印をクリックします。
2. [<b>ログオフ</b>] を選択します。

<span underline="single">デスクトップのリセット</span>

デスクトップをリセットすると、デスクトップが閉じて再起動し、保存されていないデータはすべて失われます。デスクトップをリセットするには、そのデスクトップにログインしている必要があります。

デスクトップのリセットは View Client ウィンドウから、またはデスクトップ本体内から実行できます。

View Client ウィンドウからデスクトップをリセットするには
1. View Client ウィンドウで、リセットするデスクトップの横にある下矢印をクリックします。
2. [<b>リセット</b>] を選択します。
3. 確認のために [<b>リセット</b>] をクリックします。

デスクトップ内からデスクトップをリセットするには
1. <b>&lt;Ctrl&gt; + &lt;Alt&gt; + &lt;Del&gt;</b> を押します。
2. [<b>リセット</b>] をクリックします。

<span underline="single">デスクトップ リストの更新</span>

デスクトップのリストを更新し、View 接続サーバから最後にリストを取得してからの変更を表示できます。これを行うには、<b>&lt;F5&gt;</b> キーを押します。

<span underline="single">连接到桌面</span>



连接 View 连接服务器并登录后，您就可以连接虚拟桌面。View Client 窗口中显示了您有权使用的桌面列表，以及每个桌面的当前状态。



要连接到某个桌面：

1. 从 View Client 窗口中选择所需的桌面。

2. （可选）从<b>显示</b>下拉菜单中选择一个选项丅全屏模式用整个显示器屏幕显示桌面。如果您选择 [Full Screen（全屏幕）] 并且具有多个显示器，请将桌面选择窗口拖到您想在其中显示桌面的显示器中。

3. （可选）您可以根据 View 管理员所做的系统配置，选择相应的显示协议。单击向下箭头，选择<b>协议</b>，然后选择 Microsoft RDP 或 PCoIP。有关详细说明，请与您的 View 管理员联系。

4. 双击列表中的桌面，或单击<b>连接</b>，或者单击向下箭头并选择<b>连接</b>。



您所做的选择将作为下次连接时的首选项。



<span underline="single">回滚桌面</span>



您可以回滚在另一个主机上检出的虚拟桌面?回滚检出的桌面会放弃对该桌面所做的更改，未备份到服务器上的数据都将丢失。



要回滚某个桌面：

1. 单击您要回滚的已检出桌面旁边的向下箭头。

2. 选择<b>回滚</b>。



远程主机上的检出操作将取消，您此时可以在本地主机上登录该桌面?



<span underline="single">断开桌面连接</span>



如果您断开与虚拟桌面的连接，也仍会处于登录状态。桌面中运行的程序将保持打开状态。您可以随时重新连接到该桌面并恢复会话。



要从桌面内断开连接：

1. 保存桌面上打开的文件。

2. 从桌面上的<b>开始</b>菜单中选择<b>断开连接</b>，或者按 <b>Ctrl+Alt+Del</b> 键并单击<b>断开连接</b>。



<span underline="single">从桌面中注销</span>



如果您从虚拟桌面中注销，当前的会话将会终止，所有程序都将关闭，任何未保存的数据都会丢失。您可以从桌面内注销，也可以从 View Client 窗口注销。



要从桌面内注销：

1. 保存桌面上打开的文件。

2. 从<b>开始</b>菜单中选择<b>注销</b>。

3. 在确认对话框中，单击<b>注销</b>。

如果您尚未保存打开的文件，系统将提示您进行保存。



要从 View Client 窗口内注销：

1. 单击您要回滚的已检出桌面旁边的向下箭头。

2. 选择<b>注销</b>。



<span underline="single">重置桌面</span>



重置桌面时，该桌面将关闭并重新启动，任何未保存的数据都将丢失。您必须登录桌面才能对其进行重置。



您可以从 View Client窗口或桌面本身重置桌面。



要从 View Client 窗口中重置桌面：

1. 在 View Client 窗口中，单击要重置的桌面旁边的向下箭头。

2. 选择<b>重置</b>。

3. 单击<b>重置</b>进行确认。



要从桌面内重置桌面：

1. 按 <b>Ctrl+Alt+Del</b> 键。

2. 单击<b>重置</b>。



<span underline="single">刷新桌面列表</span>



您可以刷新桌面列表，以显示自上次从 View 连接服务器获取桌面列表以来所做的更改。要执行刷新操作，请按 <b>F5</b> 键。
